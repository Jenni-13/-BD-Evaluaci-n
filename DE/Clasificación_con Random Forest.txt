1. Justificación del algoritmo
Se eligió Random Forest como algoritmo de clasificación porque:
* Es un método basado en árboles de decisión que combina varios árboles para mejorar la precisión y reducir el sobreajuste.

* Funciona bien con datasets que contienen varias características numéricas y categóricas, como breast_cancer.csv.

* Permite obtener importancia de las variables, lo que ayuda a interpretar cuáles características influyen más en la predicción.

* Es robusto ante datos desbalanceados, especialmente usando class_weight="balanced".
2. Descripción del diseño del modelo
El modelo se construyó con los siguientes pasos:
   1. Carga y exploración de datos
   * Se cargó el dataset breast_cancer.csv y se verificaron dimensiones, tipos de datos y valores nulos.
   2. Preparación de datos
   * Se definió la columna objetivo diagnosis y se codificó como M=1 (maligno) y B=0 (benigno).
   * Se eliminaron columnas no informativas como id.
   3. Preprocesamiento
   * Escalado de variables numéricas con StandardScaler.
   * Pipeline de preprocesamiento para integración directa con el modelo.
   4. Modelo Random Forest
   * Se utilizó RandomForestClassifier con parámetros ajustables:
   * n_estimators: número de árboles
   * max_depth: profundidad máxima de cada árbol
   * min_samples_split y min_samples_leaf: control de crecimiento de los nodos
   * max_features: número de características a considerar en cada split (None, sqrt, log2)
   5. Búsqueda de hiperparámetros

      * Se aplicó RandomizedSearchCV con validación cruzada estratificada (StratifiedKFold) para optimizar el rendimiento.
